# 面试模拟Agent系统 - 界面原型设计

## 文档信息
- **版本**: v1.0
- **日期**: 2026-03-12
- **状态**: 调研阶段
- **角色说明**: 用户=面试官（提问者），Agent=面试者（回答者）

---

## 1. 设计原则

### 1.1 设计理念
- **简洁专业**: 界面设计简洁大方，突出专业性
- **清晰层级**: 信息层级分明，用户易于理解
- **即时反馈**: 所有操作提供即时反馈，增强交互感
- **专注内容**: 减少视觉干扰，让用户专注于面试内容
- **角色明确**: 清晰区分面试官（用户）和面试者（Agent）

### 1.2 色彩系统

```
主色调:
├── Primary: #1a365d (深蓝) - 主要按钮、强调元素
├── Primary Light: #2c5282 - 悬停状态
├── Primary Dark: #0d1b2a - 激活状态

辅助色:
├── Secondary: #4a5568 (灰色) - 次要按钮、边框
├── Success: #38a169 (绿色) - 成功状态
├── Warning: #d69e2e (黄色) - 警告状态
├── Error: #e53e3e (红色) - 错误状态
├── Info: #3182ce (蓝色) - 信息提示

中性色:
├── Background: #f7fafc (浅灰) - 页面背景
├── Surface: #ffffff (白色) - 卡片背景
├── Text Primary: #1a202c (深灰) - 主要文字
├── Text Secondary: #718096 (中灰) - 次要文字
├── Border: #e2e8f0 (浅灰) - 边框线
```

### 1.3 字体规范

```
字体栈:
├── 中文: "PingFang SC", "Microsoft YaHei", "Helvetica Neue", sans-serif
├── 英文: "Inter", "Helvetica Neue", Arial, sans-serif
├── 代码: "Fira Code", "Consolas", monospace

字号:
├── Display: 32px/40px - 页面标题
├── Heading 1: 24px/32px - 区块标题
├── Heading 2: 20px/28px - 卡片标题
├── Heading 3: 16px/24px - 小标题
├── Body: 14px/22px - 正文
├── Caption: 12px/18px - 辅助文字
```
